# HEARTBEAT.md - Floatweb Heartbeat

## Check These During Heartbeat

### 1. Vault Check
Read `vault/HEARTBEAT.md` and check:
- Critical overdue tasks
- Sales follow-ups needed (>48h)
- Ops deadlines approaching
- Infra system health

### 2. Notify Telegram Groups
- #fw-sales: Sales alerts, follow-ups
- #fw-operations: Task updates, deadlines
- #fw-infra: System status (weekly only)

### 3. Channel Health
- WhatsApp: Connected?
- Telegram: Connected?
- Brave API: Working?

## Actions
If critical tasks found → Send to #fw-operations
If sales follow-ups needed → Send to #fw-sales
If system issues → Send to #fw-infra + notify Chief
