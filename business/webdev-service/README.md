# Web Dev One-Pager Service

## Service Overview
Build simple, effective one-page websites for local businesses without web presence.

## Pricing
- **Base package:** €250
- **Premium (booking integration):** €350
- **Add-ons:** Custom domain setup (+€50), SEO basics (+€30)

## Target Industries
1. Restaurants/Cafes
2. Hair salons/Barbers
3. Local shops/Retail
4. Trades (plumbers, electricians, painters)
5. Small B&Bs/Guesthouses

## One-Pager Structure
Every site includes:
- Hero section (business name + tagline)
- About/Story section
- Services/Menu/Products
- Contact info + map
- WhatsApp click-to-chat button
- Hours/Availability
- Simple gallery (3-5 photos)
- Mobile-first, loads in <2 seconds

## Tech Stack
- Static HTML/CSS (no backend needed)
- Tailwind CSS for styling
- Responsive design
- Deploy to: Netlify, Vercel, or GitHub Pages (free hosting)
- Custom domain: Namecheap/Cloudflare (~€10/year)

## Delivery Time
- **48 hours** from payment to live site
- 1 revision round included

## Process
1. Find business without website
2. WhatsApp outreach with offer
3. Discovery call (15 min)
4. Collect content (photos, text, hours)
5. Build site
6. Review & revise
7. Deploy + handover
8. Invoice
