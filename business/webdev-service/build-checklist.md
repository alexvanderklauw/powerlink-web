# One-Pager Build Checklist

## Pre-Build: Discovery (15 min call)
- [ ] Business name confirmed
- [ ] Tagline/slogan
- [ ] Services/menu items + prices
- [ ] Opening hours
- [ ] Address & phone
- [ ] WhatsApp number (for chat button)
- [ ] Photos received (5-10 minimum)
- [ ] Logo (if exists)
- [ ] Color preferences
- [ ] Map embed code generated

## Build Phase (~2 hours actual work)
Choose template:
- [ ] Restaurant → `restaurant-template.html`
- [ ] Salon/Barber → `salon-template.html`
- [ ] Shop/Retail → `shop-template.html` (to be created)
- [ ] Trade/Service → `trade-template.html` (to be created)

### Replace placeholders:
- [ ] {{BUSINESS_NAME}}
- [ ] {{TAGLINE}}
- [ ] {{DESCRIPTION}} (SEO meta)
- [ ] {{HERO_IMAGE}} (best photo URL)
- [ ] {{ABOUT_TEXT}}
- [ ] {{PHONE}} / {{PHONE_DISPLAY}}
- [ ] {{WHATSAPP_NUMBER}}
- [ ] {{ADDRESS}}
- [ ] {{MAP_EMBED_CODE}}
- [ ] {{YEAR}} (auto: 2026)
- [ ] Menu items / Services list
- [ ] Gallery images
- [ ] Opening hours
- [ ] Reviews (if available)

## Polish
- [ ] Test on mobile (Chrome dev tools)
- [ ] Test on desktop
- [ ] Check all buttons work
- [ ] Verify WhatsApp click-to-chat
- [ ] Test phone number click
- [ ] Ensure <2s load time
- [ ] Add favicon

## Deploy
- [ ] Upload to hosting (Netlify/Vercel)
- [ ] Connect custom domain (if ordered)
- [ ] Set up HTTPS
- [ ] Test live URL

## Handover
- [ ] Send live URL to client
- [ ] Give them 24h for revision requests
- [ ] Make one round of revisions (included)
- [ ] Invoice for payment

## Post-Launch
- [ ] Send Google Search Console setup guide
- [ ] Offer social media integration (upsell)
- [ ] Schedule check-in in 30 days
