# SimpelSite

**Brand Name:** SimpelSite
**Tagline:** "Jouw website in 48 uur, simpelweg geregeld"
**Target:** Dutch local businesses without websites

## Design Inspiration (from SaaS sites research)

### Key Patterns from Pipedrive, Base44, Linear, Notion, Cal.com:

1. **Hero Section**
   - One clear headline (not clever, clear)
   - Subheadline explaining value in one sentence
   - Primary CTA button (contrasting color)
   - Trust indicator below CTA ("al 50+ tevreden klanten")

2. **Social Proof Early**
   - "Trusted by X companies"
   - Logo bar of recognizable clients
   - Star ratings/reviews

3. **How It Works (Steps)**
   - Base44 style: 01/02/03/04 numbered sections
   - Each step: icon + headline + 1 sentence explanation
   - Visual progression (arrow or line connecting steps)

4. **Benefit Sections (not Features)**
   - Not "We use HTML" but "Works on every device"
   - Not "Fast delivery" but "Online within 48 uur"
   - Icon + benefit + micro-copy

5. **Pricing**
   - Simple: 2-3 tiers max
   - Highlight "most popular"
   - Price + what's included + CTA

6. **FAQ Section**
   - 4-6 common questions
   - Expandable/collapsible
   - Addresses objections

## SimpelSite Pages

### 1. Home (`/`)
- Hero with headline + CTA
- Logo/trust bar
- How it works (4 steps)
- Example websites (3 thumbnails)
- Testimonials
- Final CTA section

### 2. Pricing (`/pricing`)
- 2 tiers: Basis (€250) + Compleet (€350)
- Feature comparison
- "Wat krijg je precies" explanation
- FAQ about pricing

### 3. Design Your Site (`/start`)
- Multi-step form:
  1. Business type (dropdown)
  2. Business name
  3. What do you do? (textarea)
  4. Style preference (image cards: modern/warm/professional/playful)
  5. Must-have sections (checkboxes)
  6. Contact info
- Progress bar showing steps
- "Direct een offerte" CTA at end

## Visual Direction

**Colors:**
- Primary: Deep blue (#2563EB) or Teal (#0D9488) - trust + modern
- Secondary: Soft gray backgrounds
- Accent: Orange or Coral for CTAs

**Typography:**
- Headlines: Clean sans-serif (Inter, Geist)
- Body: Same family, readable size

**Vibe:**
- Not corporate/stiff
- Not too playful/casual
- Professional but approachable
- "We make this easy for you"

## Content Tone

- Dutch language
- "Jij/jouw" not "u/uw" (informal but respectful)
- Short sentences
- No jargon ("responsive" → "werkt op telefoon én computer")
- Action-oriented ("Start jouw website" not "Meer informatie")
