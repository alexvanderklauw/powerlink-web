## TIEBEPALINGSLOGICA

### Basis Tier Mapping
```
IF reserveren = "Ja":
    tier = "Pro"
ELSE:
    SWITCH uitgebreidheid:
        "Eén overzichtelijke pagina" → tier = "Start"
        "Meerdere pagina's" → tier = "Groei"
        "Best uitgebreid" → tier = "Pro"
        "Laat jullie adviseren" → tier = "Groei"
    
    IF meerdere_regio's = "Ja" AND tier = "Start":
        tier = "Groei"
```

### Extra Pagina's Berekening
```
extra_pages = 0

IF meerdere_regio's = "Ja":
    extra_pages += aantal_regios - 1
    // Per extra regio: +€25

IF diensten_aantal > 5 (bij Groei) OR > 10 (bij Pro):
    extra_pages += overschot
    // Per extra pagina: +€25
```

### Prijsberekening
```
setup_price = tier.setup_price
monthly_price = tier.monthly_price

IF pro_email = true:
    monthly_price += 5

IF extra_pages > 0:
    setup_price += (extra_pages * 25)

IF logo_maken = true:
    setup_price += 25

IF teksten_schrijven = true:
    setup_price += 79
```
