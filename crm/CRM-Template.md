# Floatweb CRM - Google Sheets Template

## Kolommen:
| Kolom | Naam | Beschrijving |
|-------|------|--------------|
| A | Datum | Contactdatum |
| B | Bedrijf | Naam van het bedrijf |
| C | Contactpersoon | Naam van eigenaar/manager |
| D | Telefoon | WhatsApp nummer |
| E | Email | Emailadres (indien gevonden) |
| F | Adres | Locatie van het bedrijf |
| G | Niche | Restaurant/Salon/Cafe/etc |
| H | Status | Nieuw/Contact/Offerte/Deal/Lost |
| I | Laatste Contact | Datum laatste bericht |
| J | Notities | Belangrijke info |
| K | Bron | Google Maps/Instagram/etc |
| L | Prioriteit | Hoog/Middel/Laag |

## Status Opties:
- **Nieuw** - Nog niet benaderd
- **Contact** - Bericht verstuurd, wacht op reactie
- **Gesprek** - In gesprek/interesse getoond
- **Offerte** - Offerte verstuurd
- **Deal** - Klant! âœ…
- **Lost** - Geen interesse/geen reactie
